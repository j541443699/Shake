# Android摇一摇
## 功能概述
当用户晃动手机时, 会触发加速传感器, 此时加速传感器会调用相应接口供我们使用, 此时我们可以做一些相应的动画效果, 震动效果和声音效果。具体功能点: 
- 用户晃动后两张图片分开, 显示后面图片
- 晃动后伴随震动效果, 声音效果
## 相关知识点
- 加速度传感器
- 补间动画 
- 手机震动 (Vibrator)
- 较短 声音/音效 的播放 (SoundPool)
